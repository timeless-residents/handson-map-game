# ハンズオンマップゲーム

このプロジェクトは、インタラクティブなマップゲームを作成するためのものです。

## 必要条件

- Node.js
- npm

## インストール

以下のコマンドを実行して依存関係をインストールします。

```bash
npm install
```

## 使用方法

開発サーバーを起動するには、以下のコマンドを実行します。

```bash
npm start
```

ブラウザで `http://localhost:3000` を開いて、ゲームをプレイします。
